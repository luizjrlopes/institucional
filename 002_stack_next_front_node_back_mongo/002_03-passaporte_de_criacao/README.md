## Objetivo

Este diretório armazena o **PASSAPORTE_DA_CRIACAO.md** gerado após a conclusão da estrutura base do projeto.

---

## O que é o Passaporte da Criação?

É um documento que registra:

- **Contexto**: Descrição do projeto criado
- **Tecnologias Instaladas**: Frontend e Backend
- **Estrutura Criada**: Árvore de diretórios completa
- **Funcionalidades Implementadas**: Auth, validação, error handling, etc.
- **Testes de Build**: Confirmação de compilação
- **Validações**: Conformidade com dossiês
- **Observações**: Adaptações ou exceções

---

## Quando é Gerado?

O passaporte é gerado pelo **AGENTE_GERADOR_PASSAPORTE_DA_CRIACAO** após:

1. AGENTE_CRIADOR criar a estrutura
2. PLAYBOOK_PIPELINE garantir qualidade (F-Designer → Auditor → Refatorador)
3. Ambos os projetos (frontend + backend) compilarem sem erros
4. Integração HTTP funcionar corretamente

---

## Como é Validado?

O **AGENTE_VALIDADOR_PASSAPORTE_DA_CRIACAO** verifica:

- Documento segue template oficial
- Tecnologias documentadas correspondem ao código real
- Estrutura de pastas corresponde ao código real
- Funcionalidades declaradas estão implementadas
- Builds passam sem erros
- Conformidade com dossiês institucionais

---

## Resultado da Validação

**✅ APROVADO:** Passaporte válido, projeto pronto para evolução  
**❌ BLOQUEADO:** Inconsistências detectadas, necessário correção
